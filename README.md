Klarna Order Updater
====================

Requirements
---------------

* Python 2.7.x
* Latest virtualenv

Setup
------

1. Clone repo
2. cd to repo's root
3. $ virtualenv env
4. $ source env/bin/activate
5. $ pip install -r requirements.txt

Run
---

$ ./cream-klarna.py

(Will display help usage if no args are provided.)
